# Frontend (sitio público)

Este directorio contiene una landing page estática tipo clínica, pensada para correr en XAMPP.

## Cómo abrir

- Abre `frontend/index.html` directo en el navegador, o
- Con XAMPP/Apache: visita `http://localhost/ProyectoNuevo/frontend/`.

## Qué incluye

- Header con topbar y menú (responsive con drawer).
- Hero con CTA.
- Secciones: Especialidades, Equipo médico, Sedes, Seguros, Blog, Contacto.
- Modal de “Agendar cita” (demo, sin backend todavía).

## Próximo paso (opcional)

Si quieres, conecto “Agendar cita” al backend Laravel en `backend/api` para guardar la cita en base de datos (modelo `Cita`).
